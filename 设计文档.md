### 🚀 技术栈选型建议（配合代码库）

- **核心框架**：Vue 3 (Composition API) + Vite
- **样式方案**：Tailwind CSS（极速实现暗黑模式和现代布局）+ SCSS
- **3D/动效引擎**：
  - `@tresjs/core`（Vue 3 的 Three.js 包装器，写 3D 就像写 Vue 组件一样简单）。
  - 或者直接使用 **Spline**（设计好 3D 模型后，通过 `<iframe src="...">` 或内嵌代码零成本接入，超级震撼且省力）。
  - `GSAP`（处理丝滑的滚动视差动画）。
- **字体**：主标题使用英文字体 `JetBrains Mono` 或 `Fira Code`（自带代码连字效果），正文使用 `PingFang SC` / `Inter`。

### 🗺️ 页面架构与路由规划（Multi-Page 体验）

为了避免单一页面的枯燥，建议将前端切分为 4 个主要路由/区块，配合顶部高斯模糊的毛玻璃导航栏进行切换。

#### 1. 首页：`/` (代号：The Awakening / 觉醒)

- **视觉核心**：全屏暗黑背景（`#0a0a0a`），配合微弱的 CRT 显示器扫描线遮罩。
- **3D 元素**：在屏幕右侧放置一个随鼠标悬浮缓慢转动的 3D 赛博朋克服务器机柜，或者一个发光的悬浮大脑神经元节点（用 Spline 制作）。
- **信息呈现**：
  - **打字机特效**：左侧屏幕模拟 Linux 终端启动，快速闪过几行 `[ OK ] Started Zhuoruan Core Service.` 的启动日志（这里可以加入一些 Arch Linux `systemd` 风格的启动字符，极客感拉满）。
  - **Slogan（吹牛版）**：**“不要在平庸的代码里浪费天赋。卓软，重塑你的极客基因。”**
  - **CTA 按钮**：一个发着霓虹青色光芒（Neon Cyan）的 `[ Initiate Join Sequence_ ]` 按钮，鼠标悬停时伴随轻微的电流故障（Glitch）动画。

#### 2. 算力与资源枢纽：`/matrix` (代号：算力矩阵)

专门用来“吹”你们的 API 平台和底层资源，这是吸引对 AI 感兴趣新生的超级杀器。

- **排版风格**：类似科幻电影里的数据监控大屏，采用网格化（Bento Box）布局。
- **板块内容**：
  - **无限火力 API**：展示 `api.zhuoruan.xyz` 的卡片。**文案**：“内部专属大模型网关。GPT-4o、Claude 3.5 Opus 全量接入。我们不画大饼，我们直接给你算力自由。按量计费，但社团内部价近乎白嫖，让你的 Agent 跑个痛快。”
  - **硬核导师镇场**：用“游戏 Boss 级”的 UI 卡片介绍指导老师。不写无聊的头衔，写“战斗力/Tech Stack”。**文案**：“资深架构师坐镇，从理论到落地全链路降维打击，无论你是准备手写 Spring Boot 微服务，还是死磕 408 考研底层原理，这里都有最硬核的指引。”

#### 3. 技术兵器谱：`/arsenal` (代号：兵器库)

展示社团的技术方向和以往战绩，要让文科生看了觉得不明觉厉，理科生看了热血沸腾。

- **3D 交互**：页面中央是一棵可交互的“3D 技能树”或者悬浮的发光卡片阵列。
- **分类展示**：
  - **Backend (后端引擎)**：Java / 数据库 / 分布式架构。
  - **Frontend (前端交互)**：Vue3 / Three.js / UI 动效。
  - **AI & Ops (智能与运维)**：Docker 容器化部署 / Multi-Agent 工作流搭建。
- **往届战绩**：用瀑布流展示开源项目截图和获奖证明。**文案**：“你的代码不应该只躺在硬盘里，它应该在服务器上高并发运行，在国家级竞赛中拿奖。”

#### 4. 终端报名页：`/register` (代号：Root Access)

**完美对接你现有的报名网站接口。** 这是整个前端交互的高潮。

- **设计理念**：抛弃传统的白色背景长表单！把整个报名页面做成一个**全屏伪终端（Pseudo Terminal）**。
- **交互流程**：
  1. 用户进入页面，屏幕是黑的，只有一个闪烁的光标 `_`。
  2. 自动输入 `sudo pacman -S zhuoruan-member` 或 `./join_zhuoruan.sh`。
  3. 屏幕打印：“请输入你的代号 (Name):” -> 用户键盘输入。
  4. 屏幕打印：“请输入你的联络频段 (Phone/QQ):” -> 用户键盘输入。
  5. 屏幕打印：“选择你的首发武器 (意向方向: 1.前端 2.后端 3.AI):” -> 用户选择。
  6. **最终动作**：用户敲击回车，前端捕获这些数据，通过 Axios 悄悄 POST 发送到你们仓库里已有的后端报名接口。
  7. **成功反馈**：满屏绿色的代码雨（Matrix Rain）落下，弹出 `Welcome to Zhuoruan, [用户名].`。

### 🎨 细节与“加分项”建议

1. **彩蛋（Easter Egg）**：在前端代码的 `main.ts` 里写一段 `console.log`。当懂技术的新生按下 `F12` 打开控制台时，用 ASCII 字符画打印出卓软的 Logo，并附言：“既然你打开了控制台，跳过一面，报名表备注栏填入 'F12_Hacker' 即可”。
2. **自定义光标**：用 CSS 把鼠标光标变成一个十字准星或代码输入符号。
3. **音效（可选）**：在点击“报名”等关键按钮时，加入极短的机械键盘敲击声或清脆的 UI 确认音（需要注意浏览器自动播放策略，需用户先发生交互才播放）。
